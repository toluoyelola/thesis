A context aware password analyser
This project aims to provide its users will secure passwords to secure their digital assets


Libraries and APIs to be used
zxcvbn (JavaScript/Python) - Industry-standard password strength estimator
Have I Been Pwned API - Free API for checking breached passwords
Levenshtein distance libraries - To detect passwords similar to username/company name
fuzzywuzzy - a fuzzy matching libraries


Frontend: HTML + JavaScript
Backend: Python + FastAPI
